Angular Tilt Api

Below you'll find the public apis currently supported by Angular Tilt

Import Path

To use angular-tilt, you'll need to import the NgTiltModule into your project 

import {NgTiltModule} from '@geometricpanda/angular-tilt';

[ngTilt] directive

You can apply angular-tilt by applying the ngTilt directive, such as: 

<div ngTilt>
  ...
</div>

Inputs

Name Type Default Value
maxTilt number 20 The maximum number of degrees for the tilt to rotate.
perspective number 300 Sets the CSS perspective, lower numbers will transition more.
easing string cubic-bezier(.03,.98,.52,.99) CSS Easing transition on mouse enter / exit.
scale number 1 Sets the scale to increase on hover, 1 = 100%, 2 = 200%.
speed number 400 Sets the speed of the transition (in ms).
disableAxis xy undefined Disables tilting on a specific axis.
reset boolean false Sets whether the glare effect should reset ton mouse exit.
glare boolean false Adds a glare effect to the tilted element
maxGlare number 0.4 Maximum brightness of the glare
global boolean false Tilts on any mousemove, not just hover.

[ngTiltParallax] directive

You can apply a parallax effect by applying the ngTiltParallax directive, to children of the ngTilt. 

<div ngTilt>
    <div ngTiltParallax>
      ...
    </div>
</div>

Inputs

Name Type Default Value
distance number 50 Parallax distance which should be applied, greater numbers make elements appear further in front of the tilt.
